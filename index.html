Project: BARLEY BIODIVERSITY - Increasing biodiversity and productivity of barley cultivars using a sensor-based high-throughput phenotyping system in the field and association mapping with two wild barley introgression libraries

Juni 2018 - November 2020

The project was part of the initiative „Innovative Pflanzenzüchtung im Anbausystem“ (IPAS)

Tasks and methods: My task in the project was the evaluation and statistical analysis of multi-modal data from a large barley maping population.The data have been obtained

using various analytical techniques and across several years. In terms of analytical complexity, the project included handling large and diverse data structures, including:

genotype data, mass spectrometry profiles, phenotype descriptors and quantitative phenotypes, as well as analysis of the data variation and reproducibility across different

cultivation years and plant developmental stages.

Analysis of metabolic data:Identification of metabolic features in the mass spectrometry data obtained by HPLC-MS/MS, providing a comprehensive profile of semi-polar compounds accumulated in barley leaves Data normalization:Data normalization using multiple internal standards by fitting a multivariate linear model to a set of reference internal standards and removing the fitted variance from the target variables. Quality control: Quality control (QC) samples were used to estimate the data reproducibility and technical error across analytical batches and cultivation seasons. ANOVA: Analysis of variance has been performed to highlight effects of different experimental factors: e.g. genotype, batch, developmental stage. Pearson correlation analysis: Correlation analysis has been performed between measured variables using Pearson’s correlation coefficient. Homogeneity of the correlations across cultivation seasons has been analysed and stable correlations have been identified. Cluster determination using GAP statistics: Cross validation-based methods for cluster number estimation have been applied to estimate the number of stable clusters in each dataset and cultivation year. Regularized regression: Regularized regression has been used for feature selection and phenotype prediction, including ridge regression, elastic net and LASSO. Software used in frame of the project: R, Excel, Compass DataAnalysis, QuantAnalysis and MetaboScape (Bruker, Bremen) The work will be submitted for publication, hopefully within a few months
